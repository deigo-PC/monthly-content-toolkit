# Knowledgebase

Place the brand's strategy and reference documents here. The AI coding agent
reads these files when generating content to ensure accuracy, tone, and
strategic alignment.

## What goes here

| Document | What it contains |
|---|---|
| `strategy-doc.md` | Overall marketing strategy, goals, messaging pillars |
| `buyer-psychology.md` | Persona deep-dives: awareness stages, triggers, objections |
| `message-blocks.md` | Approved phrases, value props, calls to action |
| `faq.md` | Frequently asked questions and brand-approved answers |
| `past-content/` | Previous months' content plans, published posts |
| `sitemap.md` or `sitemap.xml` | List of all website URLs for internal linking |

## How to use

1. Copy relevant files from the brand's existing docs into this folder
2. The agent reads them automatically when context is loaded
3. Files can be in Markdown (.md), plain text (.txt), or HTML (.html)

## Best practices

- Keep buyer-psychology.md **up to date** — it's the single most valuable
  file for content relevance
- Add new FAQs as they come in from customer support
- Archive old content plans in `past-content/` — the agent uses them to avoid
  repeating topics
